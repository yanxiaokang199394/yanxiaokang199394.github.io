第4章 - 基于Django表单，自动生成界面

发表于2022-12-19|更新于2023-01-01|Django
|阅读量:

基于传统表单HTML开发项目，会显著增加HTML页面的代码量，比如校验用户输入的合法性，数据格式是否正确等，这增加了开发难度和调试时间。Django的From表单可以自动生成HTML组件标签，增加校验功能，极大的提高了编写代码的效率

HTML表单

表单介绍

  • HTML表单是一个包含表单元素的区域，表单的主要功能是收集用户信息，以完成人机交互

  • 表单由一个个HTML标签组成，常见的HTML标签有文本框、单选框、复选框、下拉列表框、确认按钮等

    标签 说明
    <input type=password> 密码输入框标签
    <input type=radio> 单选框
    <input type=checkbox> 复选框
    <input type=button> 按钮
    <input type=submit> 提交按钮
    <input type=reset> 重置按钮
    <input type=file> 文件上传
    <textarea> 多行文本
    <label> 显示文本
    <select> 下拉框列表
    <option> 下拉框列表中的选项

CSRF保证表单的安全

  • HTML表单想要在django中正常显示，还需要添加CSRF令牌，CSRF是django为了防止跨站请求伪造而默认开启的一项保护机制

  • 通过添加标签csrf_token放在<form></from>中来为表单添加令牌

  • 在项目settings.py中，MIDDLEWARE列表包含了所有的中间件信息

Form表单

Django框架提供了form表单，用于生成页面可用的HTML标签，用户在表单中输入数据提交表单时，框架会自动进行表单数据验证，并将数据绑定在表单对象上

​

认识表单

  • 表单类被定义在每个应用目录下的forms.py文件下，默认不存在需要手工添加
  • 表单类必须继承自Form类，在包django.forms中

  1. 常用表单字段

    表单字段 说明
    CharField 字符类型
    如：<input type=’text’>
    InterField 数值类型
    如：<input type=’number’>
    FloatField 数值类型
    如：<input type=’number’>
    DecimalField 数值类型
    如：<input type=’number’>
    ChoiceField 选择属性字段
    如：<select name=’ ‘, id =’ ‘>
    FileField 文件属性字段
    如：<input type=’file’>
    BooleanField 布尔类型字段
    如：<input type=’checkbox’>
    DateField 日期属性字段，可以自动校验日期格式
    如：<input type=’text’>
    DateTimeField 日期属性字段，可以自动校验日期时间格式
    如：<input type=’text’>
    EmailField 邮件属性字段
    如：<input type=’email’>
    URLField URL地址属性字段，key自动校验url格式的合法性
    ModelChoiceField 如果使用了该字段，则可以直接从数据库中获取数据生成下拉列表组件
    如：<select name=’ ‘, id =’ ‘>

  2. 常用表单字段参数

    字段参数 说明
    label 生成HTML中的label标签
    label_suffix label标签后的统一后缀信息
    initial 字段的初始值
    help_text 字段的描述信息
    error_messages 指定错误信息
    validators 指定的字段验证规则
    required 字段是否可以为空，默认True
    disabled 字段是否可编辑
    widget 指定字段的HTML标签样式

Django模型表单

​ Django提供了模型表单(ModelForm),它可以和模型直接关联，省略了在Form表单中定义表单字段的过程

​ Django中的ModelForm就是利用数据模型Model来简化表单开发的一种高级封装技术

认识模型表单

常见的模型表单属性

模型表单属性 含义
model 用于绑定已有的模型
fields 设置模型中的哪些字段可以显示，如果将值设为__all__，则显示全部字段，如果要显示部分字段，则将部分字段写入一个列表或者元组中
exclude 禁止将模型字段装换位表单字段，用法同fields
labels 设置表单字段的label项，以字典方式表达，字典的键位模型的字段
widgets 设置表单字段的渲染效果，以字典方式表达，字典的键位模型的字段
help_texts 设置表单字段的帮助信息
error_messages 设置表单字段的错误信息

使用AJAX提交表单

​ AJAX即Asynchronous Javascript And XML(异步javascript和XML)，是一个在2005年提出的使用现有技术集合的新技术，包括: HTML和XHTML、CSS、Javascript、DOM、XML、XSLT，以及最重要的XMLHttpRequest

​ 简单来说就是在不重载整个网页的情况下，通过后台加载数据，并在网页上进行局部刷新和显示

基于jQuery技术实现AJAX

  • 通过jQuery的ajax()方法，能够使用HTTP GET方式和HTTP POST方式从远程服务器请求文本，HTML、XML或JSON，并把这些外部数据直接加载到网页相应的元素中
