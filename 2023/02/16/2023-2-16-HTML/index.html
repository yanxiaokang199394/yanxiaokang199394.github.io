什么是HTML

  1. 网页的构成：

    • HTML：用来定义网页内容的含义和基本结构
    • CSS：用来描述网页的表现与展示效果
    • JavaScript：用来执行网页的功能与行为

  2. HTML(超文本标记语言–HyperText Markup Language)是构成 Web 世界的一砖一瓦，它是一种用来告知浏览器如何组织页面的标记语言。

    • 超文本：是指连接单个或者多个网站间的网页的链接。我们通过链接，就能访问互联网中的内容
    • 标记：是用来注明文本，图片等内容，以便于在浏览器中显示，例如 <head>、<body> 等

  3. HTML的发展

    • HTML 1.0：在 1993 年 6 月作为互联网工程工作小组（IETF）工作草案发布（并非标准）。
    • HTML 2.0：1995 年 11 月作为 RFC 1866 发布，在 RFC 2854 于 2000 年 6 月发布之后被宣布已经过时。
    • HTML 3.2：1997 年 1 月 14 日，W3C 推荐标准。
    • HTML 4.0：1997 年 12 月 18 日，W3C 推荐标准。
    • HTML 4.01（微小改进）：1999 年 12 月 24 日，W3C 推荐标准。
    • HTML5：2014 年 10 月 29 日，万维网联盟宣布，经过接近 8 年的艰苦努力，该标准规范终于制定完成，是目前最为流行的版本，提供了很多标签新特性，现代大多数浏览器已经具备了 HTML5 的支持。

HTML的组成

​ HTML页面有一系列的元素element组成，而元素需要使用标签创建

标签

​ 一对标签可以设置一段文字样式，添加图片或者超链接等。

示例：

​ 在 HTML 中，<h1>标签表示标题，那么，我们可以使用开始标签（Opening tag）和结束标签（Closing tag）包围文本内容，这样其中的内容就以标题的形式显示了

属性

​ HTML标签可以拥有属性（attribute）。属性提供了有关HTML元素更多的信息。我们只能在开始标签添加属性，属性通常以名称=值成对的形式出现

HTML整体结构

  1. <!DOCTYPE html>：该标签表示声明文档类型，规定了HTML页面必须遵从良好规则，从HTML5后，是最短的有效文档声明
  2. <html>：该标签包裹了整个完整的页面，是一个根元素(顶级元素)。其他所有元素必须是此元素的后代，每个HTML文档只有一个根元素
  3. <head>：该标签是一个容器，他包含了所有你想包含的HTML页面中但不想再HTML页面中显示的内容。这些内容包括你想在搜索结果中出现的关键字和页面描述、CSS样式，字符集声明等等。
    • <meta charset = ‘utf-8’>：这个标签是页面的元数据信息，设置文档字符集编码，能够避免页面乱码问题
    • <title>：这个标签定义文档标题，位置出现在浏览器标签上，而不是页面正文中，在收藏页面时它用来描述页面
  4. <body>：包含了文档内容，你访问页面时所有显示在页面上的文本、图片、音频、游戏等等。

块级和行内

概念

​ 在HTML中有两种重要的元素类别，块级元素和行内元素

  • 块级元素：独占一行
    • 块级元素(block)在页面中以块的形式展现。相对于其前面的内容，它会出现在新的一行，其后的内容也会被挤到下一行展现
    • 比如<p>、<hr>、<li>、<div>
  • 行内元素：行内显示
    • 行内元素不会导致换行，通常出现在块级元素中并环绕文档内容的一小部分，而不是一整个段落或者一组内容
    • 比如<b>、<a>、<i>、<span>等
    • 注意：一个块级元素不会被嵌套在行内元素中，但可以嵌套在其它块级元素中
div和span
  • <div>：一个块级元素，是通用在内容容器，并没有任何特殊语义，它可以被用来对其它元素进行分组，一般用于样式化相关的需求
  • <span>：一个行内元素，是短语内容的通用行内容器，并没有任何特殊语义，它可以被用来编组元素以达到某种样式

属性

​ 标签的属性，主要用于扩展标签的内容，属性包含了元素的额外信息，这些信息不回出现在实际内容中，但是可以改变标签的一些行为或者提供数据。属性总是以name=value的格式出现

  • 属性名：同一个标签中，属性名不能重复
  • 大小写：属性和属性值对大小写不敏感，不过W3C标准中，推荐使用小写的属性/属性值
  • 引号：双引号是最常用的，不过使用单引号也没问题

常见属性：

属性名 作用
class 定义元素类名，用来选择和访问特定的元素
id 定义元素唯一标识符，在整个文档中必须是唯一
name 定义元素的名称，可以用于提交服务器的表单字段
value 定义在元素内显示的默认值
style 定义CSS样式，这些样式会覆盖之前设置的样式

特殊字符

​ 在HTML中，字符<、>、"、'、& 属于特殊字符，它们是HTML语法自身的一部分

原义字符 等价字符引用
< &lt;
> &gt;
" &quot;
' &apos;
& &amp;
空格 &nbsp;
